프로젝트: 성적 관리 시스템
개인 프로젝트

목적:

학생들이 쉽게 성적을 관리 할 수 있게 하기 위함

스킬:

Amazon Cognito(signup, login), React js (front-end), node express (back-end and api)

내용:

사용자는 학기(2022년 1학기)와 학기 별 코스 (CPSC 314) 그리고 코스별 테스크(midterm, final etc)를 만들거나 지울 수 있음.
사용자가 테스크(ex. 프로그래밍 과제)를 만들때 그 테스크가 전체 성적에 얼마만큼 들어가는지(40% out of 100%)와 sub-tasks(3개의 프로그래밍 과제) 가 몇개있는지를 입력함.
Sub-task 의 성적을 Submit 할 때마다 테스크, 코스, 학기의 성적을 모두 계산하여 데이터를 업데이트 해줌.

조은결, Bryce, Andrew, Mogi, Joshy
